README for the SPANISH LANGUAGE PACK FOR MYBB

This language pack is meant for MyBB 1.8. If you are using a newer version of MyBB then please check if a newer language pack is available. To do so please go to http://www.mybb.com/ and click Downloads and View a list of MyBB Translations....

1. INTRODUCTION

You have chosen to use MyBB as the software for your bulletin board. We wish to thank you for that choice and hope you will like our software.

In order to satisfy as many users as possible MyBB is available in a multitude of languages. By default MyBB is installed with the American English language files, but thanks to many volunteers MyBB is also available in other languages. This language pack will allow you to use MyBB in spanish.

2. INSTALLATION

The spanish text is stored in language files, which are packed in an archive file together with this readme and the spanish translation of this readme. These language files are divided amongst several folders. With this package several image files are also distributed. These images contain the spanish translation for the buttons and the usergroup images.

This readme, the spanish readme and a file called espanol.php are in the main folder, the files containing the translations of the text for the front end are in a subfolder called espanol and the files containing the translations of the text for the admin end are in a subfolder called admin in the previously mentioned subfolder espanol. Additionally a folder called images will be present. This folder contains the images with the spanish text.

You will have to create the same folder structure on your server and place the language files in those folders. Below are extensive instructions on how to do this. We will assume you have already extracted the contents of the archive to your computer, which means the files are already available and the folder structure is as it should be. We will also assume you know how to use an FTP program to connect to your server. The tasks below should be executable with any FTP program.

  1. Open your FTP program and connect to your server.

  2. Navigate with your FTP program to the folder on your server in which you installed MyBB.

  3. Navigate to the folder on your server called inc and to the folder called languages, which will be located in the folder called inc.

  4. Copy the file espanol.php from your computer to this folder on the server. It should then be present in the folder languages.

  5. After that copy the folder called espanol from your computer to this folder on the server. This folder and its contents should then be present in the folder languages.

  6. Navigate with your FTP program to the folder on your server in which you installed MyBB.

  7. Navigate to the subfolder called images and the subfolder called groupimages (which is located in the subfolder images). You will see a subfolder called espanol.

  8. Copy this folder called espanol (which is located in the subfolder groupimages) from your computer to the folder groupimages on the server. The files team-administrator.gif, team-designer.gif, team-developer.gif, team-management.gif, team-mod.gif, team-supermod.gif, team-support.gif en team-tester.gif will be located in this folder called spanish.

  9. Navigate with your FTP program to the subfolder called install on your server.

  10. Copy the file index.php from your computer to this folder on the server.

  11. Navigate with your FTP program to the subfolder called resources in the folder install on your server.

  12. Copy the files language.lang.php, mybb_theme.xml, mysql_db_inserts.php, output.php, settings.xml, tasks.xml and usergroups.xml will be located in this folder called resources.

You have now uploaded all necessary files to the server. Read the following paragraphs for more information on how to enable the spanish language on your board.

3. USAGE

After uploading the required files you can use the spanish translation for your copy of MyBB. We will here after discuss how to use the spanish translation for the front end of the board, how to use the spanish translation for the admin area of the board and how to set the spanish translation as the default language for your board. The default language will be used by guests and users who haven't set a language preference.

  • To use the spanish language for the front end of the board, a user should go to the User CP and click Edit Options. Then he can select Español (Internacional) as "Board Language" below "Other options".

  • To use the spanish language for the admin part of the board, an administrator should go to the Admin CP and click Change below "Board Settings". At the bottom of the list he will see "Control Panel Preferences (Global)". Click the button Go next to it. Then select Español (Internacional) below "Control Panel Language" and click "Submit Changes" to submit your changes.

  • To set the spanish language as the default language for your board, you should to the "Board Settings" (Ajustes del Foro). The second category in the list will be "General Configuration" (Parámetros Generales). Click the button Go next to it. Then select Español (Internacional) below "Default Language" and click "Save Settings" (Guardar Ajustes) to submit your changes.

4. ADDITIONAL INFORMATION

MyBB is a relatively young software package. Support for multiple languages has only recently been introduced, which means you might spot a few errors in the translation. We encourage you to report these errors at SoporteMyBB. Your comments are a valuable source of information and we will use your comments to improve future translations!
